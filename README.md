# HeadHunter
Дипломный командный проект в Яндекс.Практикуме по курсу "Android-разработчик". Приложение для поиска вакансий с помощью Head Hunter API.

## Используемые технологии и инструменты
- Kotlin
- Room
- Retrofit
- OkHttp
- Kotlin Coroutines
- Koin
- Glide
- ViewBinding
- Jetpack Navigation
- RecyclerView, DiffUtil
- SharedPreferences
- Clean Architecture (MVVM)
- Github Actions

## Функционал приложения
- Поиск вакансий по названию
- Фильтрация поиска: страна, регион, зарплата, отрасль, не отоброжать без з/п, выбор минимальной з/п
- Просмотр детальной информации о вакансии
- Возможность поделиться вакансией
- Добавление вакансий в избранное
- Реализована тёмная тема
